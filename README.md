# Banking Management System (DBMS Project)

## Project Overview
This project is a web-based Banking Management System developed as part of a Database Management Systems (DBMS) course. The system automates common banking operations such as account management, deposits, withdrawals, fund transfers, and transaction history tracking.

## Features
- User Authentication (Admin & Employee)
- Customer and Account Management
- Deposit, Withdraw, and Fund Transfer
- Transaction History with filtering
- Role-based access control
- Secure database operations using MySQL

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: XAMPP (Apache)
- Version Control: Git & GitHub

## How to Run the Project
1. Install XAMPP
2. Copy project folder into `htdocs`
3. Import database using phpMyAdmin
4. Start Apache & MySQL
5. Open browser and go to `http://localhost/Bank`
